# Atmos

Atmos is a simple Android weather application built to better understand API handling in Android development.

## Purpose

This project was built to practice:
- API integration
- Retrofit for network requests
- MVVM architecture
- Handling API responses in Android

## Tech Stack

- Java
- MVVM Architecture
- Retrofit
- Android XML UI
- Material 3 (Material You)

## Description

The app fetches weather data from an external API and displays basic weather information such as temperature and weather conditions.  
This project is mainly built for learning how Android apps communicate with APIs.
